# Google Ads 投流系列 09：优化与排错清单

> 系统化排错，等于给团队装上“稳定器”。在 AI 角色扮演行业，除了常规的量化问题，还会遇到“合规驳回/敏感词/名人肖像”等特殊情况。这里给出“数据→结构→目标→创意→LP→出价→政策”的一条龙诊断手册。

## 1. 低展示/低点击（Impr/CTR 低）

可能原因：
- 匹配过窄：完全匹配占比过高，词组/广泛覆盖不足；
- 竞价不足：CPC/目标 CPA 过低；
- 素材弱：标题平淡、无关键词、无价值陈述；
- 预算太散：系列/组过多，每组量级不足；
- 政策限制：素材或 LP 存在敏感元素，系统降权或驳回。

解决：
- 扩展词组/广泛，配合否词；
- 提升预算/竞价，直到拿到足够曝光采样；
- 改写文案：关键词 + 价值 + 社证 + CTA；
- 合并低量组，集中量级；
- 自查合规：去除性暗示/露骨/名人/未成年人暗示；LP 增加安全政策链接。

## 2. 低转化（CVR 低）

可能原因：
- LP 与文案不一致：承诺与实际脱节；
- 速度慢/动线差：移动端体验不佳；
- 受众不匹配：意图偏信息获取、非体验；
- 追踪丢失：事件上报不稳定；
- 学习期被打断：频繁改价/暂停/合并系列。

解决：
- 重写 LP 首屏价值与 UI 展示；
- 优化性能与按钮可见性；
- 收紧受众/否词，增加再营销；
- 校验事件与去重；
- 稳定 7–14 天不大改。

## 3. 高 CPA

可能原因：
- 出价策略不合适：早用 tROAS，但 value 不准；
- 否词不足：低质流量占比高；
- 预算分散：量级不足导致学习效率低；
- 竞品挤压：CPC 上升；
- 合规打击：素材受限导致量价失衡。

解决：
- 回退到 Max Conv/tCPA，等 value 稳定再切 tROAS；
- 强化否词清理与长尾拓展；
- 合并系列/组、聚焦高价值类目；
- 调整地域/时段/设备，避开高竞价时段；
- 替换/修正素材与 LP 合规问题。

## 4. 低 ROAS（含毛利 ROAS）

可能原因：
- 定价与权益：试用过大但订阅权益弱；
- 退款/风控：异常高导致净收入低；
- 类目结构：低留存类目吞预算；
- value 传递：未传毛利或币种/金额错误；
- LP 承诺与实际偏差，形成承诺成本。

解决：
- 与产品梳理试用-订阅价值；
- 回传退款与风控失败单；
- 预算倾斜至高留存类目/地区；
- 服务器端统一 `value` 与 `margin_value`；
- 修正 LP 与广告承诺的一致性。

## 5. 追踪与口径问题（Ads vs GA4 差异大）

排查：
- 时区/货币不一致；
- 事件重复（双重导入/缺少 `transaction_id`）；
- 跨域断裂/UTM 丢失；
- Consent/增强转化未启用；
- 退款回冲口径不同。

修复：
- 统一配置；
- 选用 Ads 本地或 GA4 导入其一；
- 加强跨域与参数保留；
- 启用增强转化；
- 每月对账与离线回传。

## 6. 学习期被打断（波动大）

- 控制改动幅度 ≤ 20%；
- 合并并稳定结构；
- 延长观察窗 7–14 天；
- 每次只改一个变量。

## 7. 政策与合规（行业特有高频）

常见驳回：
- 成人/性暗示/裸露（文字或视觉）；
- 暗示未成年人角色；
- 名人/品牌角色侵权；
- 误导性承诺（线下关系/成人服务）；
- 不透明的订阅与退款说明。

解决：
- 全量 SFW：UI 截图与抽象插画，不出现性化人体；
- 明确“虚拟角色/故事共创/匿名安全”，禁止成人暗示；
- 不使用名人/品牌角色；
- LP 首屏展示安全政策与订阅说明；
- 建立创意与 LP 的合规清单（上线前自检）。

## 8. 自动化规则与脚本

- 告警：花费无转化/CPA 超标/ROAS 低/匹配率骤降/频次过高；
- 止损：暂停高 CPA 的广告组/关键词；
- 放量：表现优胜的组自动提 10–15% 预算；
- 黑名单：GDN 低效位置自动排除；
- 报表：每日报送“搜索词健康度与否词变更”。

## 9. 快速 SOP

1) 先查数据与归因；
2) 再看结构与预算分布；
3) 搜索词与否词、创意与 LP 一致性；
4) 校正出价与学习期；
5) 处理合规驳回；
6) 自动化规则兜底。
