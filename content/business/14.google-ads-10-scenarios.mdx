# Google Ads 投流系列 10：AI 角色扮演的典型场景打法

> 不同商业形态与受众圈层，对投放组合与节奏的要求不同。本篇选取 5 个与 AI 角色扮演高度相关的场景：Web 订阅、App 订阅、多语言出海、SFW 恋爱陪伴、二次元/动漫风，给出系列组合、指标与 SOP。

## 场景一：Web 订阅/内购（H5/桌面优先）

- 目标：订阅与消息包内购；
- 系列组合：NonBrand Search（主力）+ Brand Search（护城河）+ 再营销（行为×时间窗）+ YouTube（演示/口碑）+ PMax（成熟后放量）；
- LP：首屏 UI 演示 + 价值/社证/安全政策 + 免费开始；
- 追踪：`purchase` + `start_chat` + `create_character` + `first_session_len_bin`；
- 出价：Max Conv → tCPA → tROAS（毛利）；
- 指标：CVR、CPA、毛利 ROAS、首会话长度、D1/D7 回访；
- SOP：
  1) 搜索以词组/广泛+否词起步，拿到稳定样本；
  2) 再营销分层（1–3/4–7/8–30 天）；
  3) 每周替换低效素材与否词清理；
  4) 订阅/退款离线回传；
  5) 以毛利 ROAS 与 Payback 做放量护栏。

## 场景二：App 订阅（iOS/Android）

- 目标：安装 → 首会话 → 订阅/内购 → 续费；
- 系列组合：App Campaigns（AC）+ Brand Search + NonBrand Search（承接“roleplay/companion chat app”）+ 再营销（可用时）+ YouTube；
- 素材：短视频演示“角色选择 → 对话 → 记忆提示”，SFW；
- 追踪：Firebase/AppsFlyer + SKAN（iOS）；深度事件映射到 CV；
- 出价：AC 先安装，再深度事件（订阅/首会话长度）；Search/YouTube/PMax 随信号成熟切换 tCPA/tROAS；
- 指标：CPI、激活率、首会话长度、订阅率、留存、LTV；
- SOP：
  1) AC 启动后 7–14 天看 CVR 与深度事件；
  2) Search 品牌保护 + 非品牌承接；
  3) SKAN 转化值映射“会话长度/创建角色/付费”；
  4) 订阅/续费离线回传，形成闭环；
  5) App 与 Web 报表打通，避免重叠。

## 场景三：多语言出海（EN → JP/KR → ES/DE）

- 目标：按地区/语言扩量，保持 ROAS 与留存；
- 组合：各语种复制“Search + Brand + 再营销 + YouTube + PMax”的四分法；
- 本地化：关键词采用本地自然说法，LP 与客服语言适配；
- 指标：各语种的订阅率、退款率、留存、客单价、毛利 ROAS；
- SOP：
  1) 每开一国先用 Search 非品牌小预算探测；
  2) 看留存与订阅率达标再开 PMax/YouTube；
  3) 高 LTV 市场（JP/KR）可加权预算；
  4) 每月复审政策合规与敏感词库（本地化版本）。

## 场景四：SFW 恋爱陪伴（Companion）

- 目标：高 D1/D7 回访与订阅率；
- 组合：Search（companion/friend/chat）+ YouTube（故事与情绪价值）+ 再营销（温柔召回）+ PMax（成熟后）；
- 文案：温暖/治愈/匿名安全/随时可聊；
- 素材：UI 与对话内容为主，避免真人影像引发误解；
- SOP：
  1) 关键词强调 companion/friend/safe/匿名；
  2) LP 放置安全政策与举报入口；
  3) 再营销按时间窗分层，频控严格；
  4) 关注退款与投诉指标，优化预期管理。

## 场景五：二次元/动漫风（Anime/VTuber 氛围）

- 目标：覆盖 ACG 受众、提升角色收藏与订阅；
- 组合：Search（anime roleplay/anime chat/二次元 ai 聊天）+ YouTube（UI 演示 + 插画）+ 再营销 + PMax；
- 合规：避免名人/品牌角色/侵权；插画 SFW；
- SOP：
  1) 角色合集页作为 LP（恋爱/校园/奇幻/科幻…）；
  2) 每周上新角色与主题；
  3) 再营销素材强调“你离角色只差一步”；
  4) 以收藏/创建/会话长度作为质量指标，带动订阅。

## KPI 与护栏（通用）

- 业务：毛利 ROAS ≥ 目标；Payback ≤ 目标天数；
- 中间：CVR ≥ 基线；首会话长度 ≥ 基线；角色创建 ≥ 基线；
- 结构：品牌占比 ≤ 30%；高 LTV 语种预算优先；
- 风险：退款率 ≤ 阈值；政策驳回率 ≤ 阈值。

## 快速清单

- 系列：Search 品牌/非品牌、再营销、YouTube、PMax；
- 创意：UI 演示 + 文案四件套（关键词+价值+社证+CTA）；
- LP：首屏价值/按钮/安全政策可见；
- 追踪：订阅/内购 + 会话质量事件；
- 合规：SFW、禁止名人侵权；
- 看板：分语种/类目/时间窗管理。
