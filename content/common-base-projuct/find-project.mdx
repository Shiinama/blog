# 另外一种生意：垂类导航站

起因是要录一下最简AI项目的做项目视频。但这段时间因为Google Trend实在没有合适的关键词（被Google AI生态刷屏了），所以因此我介绍另外一种可以做的小生意：垂类导航站生意。

## 这个生意是怎么做的那？

市场上现在的划分是分为大导航站和垂类导航站。

他们的本质都是TO B的，市场上的企业为了竞争Google的排名和流量，而垂类导航站会简单一些。

1. **SEO价值**：为企业提供高质量的反向链接（backlinks），帮助提升他们在搜索引擎中的排名
2. **流量引导**：将特定领域的潜在客户引导至企业网站
3. **品牌曝光**：在特定垂直领域内增加企业的品牌知名度

## 如何建立一个成功的垂类导航站？

### 1. 选择合适的垂直领域

选择一个你熟悉或者有兴趣的垂直领域非常重要。理想的领域应该：

- 有足够的企业愿意为排名和流量付费
- 竞争不太激烈（避开已经有大型玩家的领域）
- 有持续的搜索需求

### 2. 盈利模式

- **付费展示位置**：首页或分类页面的顶部位置
- **特色列表**：突出显示的企业或产品列表
- **付费评测**：深度评测特定企业或产品
- **广告位**：传统的展示广告

### 3. SEO策略

- **一个相关域**： 找出一个和想做的品类相关的域
- **关键词研究**：找出目标领域的高价值关键词
- **内容优化**：输出高质量文章，切记这里和工具站的思路不太一致，所有的文章最好是AI+人修，推荐使用`gemini`的`Deep Research`功能，文章质量会非常的高。
- **技术SEO**：确保网站速度快、结构合理、移动友好
- **外链建设**：开始发外链和买外链

> 最终是当然是要尽可能做到第一页的排名，但是因为相关外链的因素，总是有企业不差钱直接来买你的广告位和外链的。

## 实际案例分析

我们以`https://nsfw.tools/`这个网站为例子，因为我有他们的价目表~，而且他们是一个很经典的Google排名靠后都赚钱的例子。

从上面的价目表可以看出，这个垂类导航站提供了多种广告位和合作方式，我帮大家翻译一下：

1. **分类页面横幅广告**

   - 价格：$750/月，三个月享10%折扣
   - 位置：所有分类页面
   - 规格：1280x300像素的横幅
   - 展示方式：轮播展示（最多可展示5个横幅）
   - 目前仅剩1个位置

2. **博客底部横幅广告**

   - 价格：$550/月，三个月享10%折扣
   - 位置：所有博客文章底部
   - 规格：1280x300像素的横幅
   - 展示方式：同样采用轮播形式
   - 目前有2个位置可用

3. **首页横幅广告**

   - 价格：$650/月，三个月享10%折扣
   - 位置：网站首页
   - 规格：1200x675像素的大型横幅
   - 展示方式：轮播展示
   - 目前仅剩1个位置

4. **产品页面横幅广告**

   - 价格：$950/月，三个月享10%折扣
   - 位置：所有产品评测页面底部
   - 规格：1200x675像素的横幅
   - 展示方式：轮播展示
   - 目前有2个位置可用

5. **专属博客文章**

   - 价格：一次性$800
   - 内容：为客户产品发布专属博客文章
   - 要求：至少700字，包含1个dofollow链接
   - 附加服务：可额外付费请他们代写

6. **套餐（一年期）**

   - 价格：一次性$1000
   - 展示期限：一年
   - 位置：产品列表页面和相关分类
   - 特点：无需加入联盟计划或提供返利链接

7. **套餐（永久）**

   - 价格：一次性$2500
   - 展示期限：永久
   - 位置：同上
   - 特点：同样无需加入联盟计划

**估算收入**

假设该网站所有广告位都售出：

- 月收入：$750 + $550×2 + $650 + $950×2 = $3,850/月
- 加上每月平均2篇付费博客文章：$1,600/月
- 每月平均1个一年期付费列表：$1,000/月
- 每季度1个永久付费列表：约$833/月

**总计潜在月收入**：约$7,283/月，即每年约$87,400

## 我的一些想法

这就是很典型的，低成本精力开发，高收入的例子了。

所以我的想法是录一个做AI占卜垂类系列的导航站。这其实也帮我确认的基建的有效性：I18n的国际化、文章系统（我发现差一个上传Word文档）、传图、无成本部署。

因此我就买了个域名`destinyai.tools`这是属于AI大类下的占卜垂类，我还提前做了很多的行业资料功课，作为录视频的项目。

但其实这还不够垂，大家如果要做可以做更垂类的，比如占卜垂类下的占星、卡罗牌、易经、数字命理、卢恩符文等，真的是非常不错的练手项目了。
