## 网站主要页面SEO优化样板

很多时候我会觉得Google的搜索引擎在侮辱产品，因为在某种程度上来说它对于`中小型产品`来说必要条件其实就是`字多`，这真的很蠢。

> 大型产品反而没这种顾忌了，专注产品体验就好，他们本身的网站权重和流量就足够爬上搜索引擎中更高的位置。

产品和UI的终极哲学是，不用字，用功能和视觉表达就吸引用户。但是搜索引擎却要求你用大量的文字来证明你的价值。这也是我们在搜索引擎的高排位网站看到大量的文字的核心原因。

所以工具页就需要：核心功能、文字、图片、FAQ

```typescript
return (
<>
  <div className="mb-8 text-center">
    <h1 className="text-primary mb-4 text-4xl font-bold md:text-5xl">{t('title')}</h1>
    <p className="text-muted-foreground mx-auto max-w-2xl text-lg">{t('description')}</p>
  </div>
  <SpeechGenerator />
  <div className="mt-20">
    <ContentSections sections={contentSections} className="mt-12" />
  </div>

  <div className="mt-20">
    <h2 className="text-primary mb-8 text-center text-3xl font-bold">{t('faq.title')}</h2>
    <FAQSections faqs={faqSections} />
  </div>
</>)
```

页面是这样，我一般会在document返回中，看看文字和结构的完整性。（这也是爬虫爬的东西）

![](https://ik.imagekit.io/ixou4q6nu/seo-finally-page.png?updatedAt=1754216906763)

很简单，本质就是填充带关键词的文字并保证内容对用户是有用的，像首页/内页其实都是一样的逻辑。

## 盘盘项目

项目到这就算完结撒花了，我们来盘点一下整个项目做了哪些事情：

1. 一整套Cloudflare部署体系，项目的CI/CD。
2. NextAuth的登录。
3. 整套用户体系。
4. 整个文件上传、桶。
5. 自动跑项目I18n翻译并生成翻译文件。
6. AI文章的生成系统，跑通一整个生成AI文章跑流量的方式。
7. AI音频/文字/图片市面主流跑通。
8. 整个Token消耗体系。
9. 付费/订阅的体系。

这可能是世界上成本最低，开发最快，最通用的海外AI类项目基建了。从技术选型上，我们用Cloudflare生态避开了的后期高额账单，用Next.js Server Actions让前后端写在一起（这是TS最快的写法），搞了个零成本的登录系统不用担心后期迁移数据库的麻烦。支付选的是个人就能申请的方案，门槛低。

AI能力这块覆盖了文/音/图的主流API，虽然要做好产品肯定需要对AI更深入的探索，但现在这些API组合已经够实现市面上大部分功能了。I18n和批量博客都是只花API成本或者免费的方案。还有多版本架构，不同类型的项目都能用。

而剩下的只需要装上这艘小船的`发动机`，去面对海上的风浪。

## 吐槽时间

### 好多的内容你就粗略的展示一下，直接就跳过了，这谁看得明白啊？

我的判断是，如果这些事情都是能通过文档得到的信息，或者是自己简单思考一下就能想明白的事情，我实在没有理由从文档扒过来，再写一遍。

这对大家其实用处也不大，毕竟并不是宝宝培训班，真正自己在做项目的事情只会更难，你要去主动收集更多的信息，去自己想明白一件事情为什么是这么运转的。

举个很简单的例子：比如项目的支付永远是核心的大问题，然后你有一些限制条件（比如没有海外的公司主体，要做的话要花4000块钱），你就需要去市场不断地找信息，看文档，去找那种不需要海外公司主体支付的方式。

可能大家在以前的开发中，就是公司帮你准备好，给你提供API Key文档，然后开发完善，但你自己做事显然不是这样的。

> 别人告诉你和自己去感受得到的东西是不一样的。

### 为啥不做国内市场市场？

这问题被人问太多次了。

国内做东西最难的点对于我来说是在于：你可以很容易的在一个项目中获得很多流量，但流量的变现和商业流程的跑通非常的痛苦，这跟印度市场的表现几乎是一致的。

当然我完全放弃做国内的东西的还有很多原因：AI行业国内很恶性循环没法做现金流，大家基本是靠融资的钱在烧。投流在大厂的AI自留地投不出去。国人太卷了，做一个东西马上就被抄。人在国内做东西是受限的等等...
