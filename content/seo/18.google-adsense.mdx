# Seo最简单的赚钱方式-广告

当我们已经讲完有关基本功的部分，就开始直接讲毕竟好入门赚钱的部分吧。

## Google Adsense简介

如果你之前没有用过Google Adsense这些Google全家桶，那么一定要分清楚Google Adsense和Google Ads是两个东西，Adsense是为你赚钱的，而Ads是要你花钱的地方。

- **Google Adsense**: 网站主通过在自己网站上展示广告来赚钱的平台
- **Google Ads**: 广告主投放广告的平台，需要支付费用

## 注册Google Adsense账号

首先进入到[Google Adsense官网](https://adsense.google.com/start/)

![Google Adsense官网首页](https://ik.imagekit.io/ixou4q6nu/adsense-1.webp)

点击右上角开始使用，然后登陆你的谷歌账号

会让你填写两个信息：`Your Site`、`Your Country`。

`Country`就正常填你本人所在的国家就行了。我在中国，所以我选择了China。不需要像Stripe或者别的平台，要填香港或者国外，就填写你本人所在的国家就好了，因为未来是要绑定收款银行卡的。

> **重要提示**: `Google Adsense`支持国内的银行卡收款。

![填写网站和国家信息](https://ik.imagekit.io/ixou4q6nu/adsense-2.webp)

填写完毕后点下一步

## 设置Google Adsense账号

接下来就会进入到Google Adsense的主界面了，你可以把侧边栏每个按钮都点开看一看，都有什么内容。

### 完善收款信息

这里我们先点击中间三张卡片最左边，完善我们的收款信息

![完善收款信息](https://ik.imagekit.io/ixou4q6nu/adsense-3.webp)

Account Type就根据你的真实情况，公司就填公司，个人就填个人。

### 添加网站

然后我们再次回到Adsense的首页，点击Submit Site（添加网站）

![添加网站](https://ik.imagekit.io/ixou4q6nu/adsense-4.webp)

## 设置广告代码

Google Adsense有一个比较方便的地方，就是它的自动广告。基本上你只需要获取它的代码，不需要专门给每个广告设置固定的位置，它会自动寻找合适的地方插入广告，比较方便。

![自动广告设置](https://ik.imagekit.io/ixou4q6nu/adsense-5.webp)

### 广告代码的获取与安装

1. 在AdSense控制面板中，点击"广告"选项卡
2. 选择"按网站"，然后点击你的网站
3. 点击"获取代码"按钮
4. 复制生成的JavaScript代码
5. 将代码粘贴到你网站的`<head>`标签中

```html
<script
  async
  src="https://pagead2.googlesyndication.com/pagead/js/adsbygoogle.js?client=ca-pub-XXXXXXXXXXXXXXXX"
  crossorigin="anonymous"
></script>
```

### 广告审核过程

添加代码后，Google需要对你的网站进行审核，这个过程通常需要1-2周时间。审核期间，请确保：

1. 你的网站内容符合Google AdSense政策
2. 网站可以正常访问
3. 网站有足够的原创内容
4. 网站导航清晰，用户体验良好

## 广告收益优化技巧

1. **内容质量**: 高质量、原创的内容能吸引更多访问者，提高广告点击率
2. **广告位置**: 虽然是自动广告，但你可以在AdSense后台设置广告展示的偏好
3. **移动端优化**: 确保你的网站在移动设备上表现良好
4. **页面加载速度**: 优化网站加载速度，提高用户体验和广告展示机会
5. **定期分析**: 使用AdSense分析工具查看哪些页面表现最好，据此调整内容策略

## Adsterra - 另一个广告平台选择

除了Google AdSense外，Adsterra也是一个不错的广告平台选择，特别是当你的网站还不符合AdSense的要求时。

### Adsterra简介

Adsterra是一个全球性的广告网络平台，成立于2013年，为网站主和广告主提供多种广告形式。相比Google AdSense，Adsterra的审核标准相对宽松，对网站的要求也较低，适合新站和小站主使用。

### Adsterra的主要优势

1. **快速审核**：通常在24小时内完成网站审核
2. **多种付款方式**：支持PayPal、WebMoney、Bitcoin等多种付款方式
3. **低门槛**：对网站流量和内容的要求较低
4. **高填充率**：几乎100%的广告填充率
5. **多样化广告形式**：包括弹窗广告、原生广告、横幅广告、直接链接等

### 如何注册Adsterra

1. 访问[Adsterra官网](https://publishers.adsterra.com/signup)
2. 填写基本信息并创建账户
3. 验证邮箱
4. 添加你的网站
5. 等待审核通过（通常24小时内）

### Adsterra广告代码安装

1. 登录Adsterra账户
2. 在控制面板中选择"网站"
3. 点击"获取代码"
4. 选择你想要的广告形式
5. 复制生成的代码
6. 将代码粘贴到你网站的适当位置

### Adsterra常用广告形式

1. **Social Bar**：移动设备上的粘性广告，展示在屏幕底部
2. **Native Ads**：与网站内容融为一体的原生广告
3. **Popunder**：在用户点击网站内容时在后台打开的广告
4. **Banner**：传统的横幅广告，可以放置在网站的不同位置

### Adsterra与AdSense的对比

| 特点     | Adsterra                | Google AdSense |
| -------- | ----------------------- | -------------- |
| 审核速度 | 快（24小时内）          | 慢（1-2周）    |
| 审核难度 | 较低                    | 较高           |
| 广告质量 | 中等                    | 高             |
| 收益潜力 | 中等                    | 高             |
| 付款门槛 | $5-$100（根据付款方式） | $100           |
| 广告形式 | 多样（包括弹窗）        | 多样（无弹窗） |

### Adsterra使用技巧

1. **合理选择广告形式**：不同的广告形式有不同的收益和用户体验，需要平衡
2. **避免过度使用**：过多的广告会影响用户体验，导致访客流失
3. **测试不同位置**：尝试不同的广告位置，找出最佳效果
4. **关注数据分析**：定期查看广告表现数据，优化策略
5. **遵守规则**：虽然Adsterra相对宽松，但仍需遵守其广告政策

### 何时选择Adsterra而非AdSense

1. 你的网站是新建的，内容还不够丰富
2. 你的网站内容属于AdSense不接受的类别（如成人内容、赌博等）
3. 你希望快速开始获得广告收入
4. 你想尝试更多样化的广告形式，包括弹窗广告
5. 你的网站被AdSense拒绝或封禁

Adsterra是初创网站或不符合AdSense要求的网站的良好选择，但随着网站的成长和内容的完善，建议同时申请AdSense，因为长期来看，AdSense通常能提供更高的收益和更优质的广告。
