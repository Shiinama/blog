# 点击深度、页面布局与流量权重的关系

这一章的本质是对[第二章网页结构](https://articles.zsxq.com/id_n1ewnq6tw21o.html)的进一步解释，也是为了解释在最简项目中的布局考虑。

在SEO中，点击深度(Click Depth)和流量权重(Traffic Weight)是两个密切相关且对网站排名有重大影响的指标。

## 什么是点击深度？

点击深度指的是用户从网站首页到达特定页面所需的点击次数。例如：

- 首页的点击深度为0
- 首页直接链接的页面点击深度为1
- 需要通过首页→分类页→内容页的路径访问的页面点击深度为2

点击深度是衡量网站结构扁平化程度的重要指标，也是搜索引擎蜘蛛爬取网站内容难易程度的直接反映。

## 什么是流量权重？

流量权重是搜索引擎为网页分配的一种价值评估，反映了该页面在整个网站中的重要性。流量权重受多种因素影响，包括但不限于：

- 页面接收的内部链接数量
- 链接来源页面的权重
- 页面的外部链接质量
- 页面内容的相关性和质量
- 用户行为信号（如停留时间、跳出率）

## 点击深度如何影响流量权重

### 1. 权重传递衰减

搜索引擎在传递页面权重时遵循"距离衰减"原则。当点击深度增加时，从首页传递的权重会逐级减少：

```
首页权重 → 一级页面(衰减20-30%) → 二级页面(再衰减20-30%) → ...
```

这意味着点击深度越大，页面从网站整体获得的权重就越小。

### 2. 爬虫抓取频率

搜索引擎蜘蛛通常会优先爬取点击深度较低的页面，并且爬取频率也更高。点击深度过高的页面可能面临以下问题：

- 被爬取的频率降低
- 内容更新后被索引的延迟增加
- 在某些情况下甚至可能不被完全爬取

### 3. 用户体验因素

点击深度也直接影响用户体验：

- 深度较低的页面通常获得更多自然流量
- 用户更容易找到并访问这些页面
- 良好的用户体验信号（如较低的跳出率）会正向反馈到页面权重

## 页面布局与链接位置对权重的影响

在讨论点击深度时，一个常被忽视但实际上非常重要的因素是链接在页面中的位置。即使是相同点击深度的页面，其获得的流量和权重也可能因链接位置不同而存在显著差异。

### 页面区域权重差异

搜索引擎会根据链接在页面中的位置给予不同的权重考量：

1. **Header(页头)区域链接**：通常获得最高权重

   - 位于页面顶部，是用户首先看到的区域
   - 几乎存在于网站的每个页面
   - 搜索引擎视其为网站最重要的导航结构

2. **Footer(页脚)区域链接**：获得中等偏高权重

   - 虽然位于页面底部，但同样出现在大多数页面
   - 通常包含网站的核心页面链接
   - 搜索引擎认为页脚链接代表网站的重要结构

3. **主内容区域链接**：权重取决于上下文

   - 内容中自然出现的链接通常获得较高权重
   - 相关性强的上下文链接被视为更有价值
   - 编辑精心挑选的链接比自动生成的相关文章链接权重更高

4. **侧边栏链接**：通常权重较低
   - 经常用于展示广告或次要内容
   - 可能在不同页面间变化
   - 搜索引擎将其视为辅助导航

> 网站的内页流量排名几乎一定会遵循在布局、点击深度、链接位置的综合计算出的权重高低（除非有外力改变，某个页面投广告、买外链、社媒爆炸），所以要尽可能利于这种权重，突出核心页面。

## 最佳实践

在布局中每个页面都由`header`、`main`、`footer`三标签构成最基础的权重表达。

`Header`和`Footer`在整个网站中保持一致，这种一致的结构向搜索引擎表示这里面的链接很重要。

在`Header`里是网站的主导航以及一些主推的功能性导航，在`Footer`里是网页里需要强推的文章系列（想抢的词、大搜索量的词的文章或者落地页）。

把不够重要的页面（可以是文章、落地页）统统压缩到二级页做更深的点击深度，为`Footer`或者`Header`预留更多的版面。

而对于`Main`里面的内容，就自然而然的完成主功能即可。
