# 项目的SEO冷启动

SEO冷启动是指新网站或新项目从零开始建立搜索引擎可见性的过程。这个阶段通常比较困难，因为搜索引擎对新网站的信任度较低，需要通过一系列策略来提升网站的权重和可见度。

在完成编码之后，项目需要做一个SEO的冷启动，步骤有：增加网站收录 -> 购买外链 -> 内容优化 -> 持续监测与调整

节奏和进度是这样的：精致AI+人修内容100篇或纯AI垃圾5000篇后，开始做`外链`，第一阶段的目标是把站的权重做到15+，至少收录域在300+（这个数量如果网站没有一点流量会需要更多），而在启动阶段做外链有两种方式：

1. 比较慢和受控制的方式是：一家家去慢慢添加和找外链 + 社群小伙伴们做了站有一些外链积累。
2. 比较块的方式是：直接买，大概5000块，能把站的外链在一个月内就干到15+。

## 提交站长工具阶段

当我们完成了所有网站的主要功能后，我们需要根据目前要做的语言去添加不同的站长工具。以下是需要提交的主要搜索引擎站长工具：

- Google Search Console (全球): https://search.google.com/search-console - 最重要的站长工具，提供索引状态和搜索表现数据

- Bing Webmaster Tools (全球): https://www.bing.com/webmasters/ - 微软的搜索引擎，同时为Yahoo提供搜索结果

- Baidu Webmaster Platform (中国): https://ziyuan.baidu.com/site/ - 中国最大的搜索引擎，需要中国手机号验证

- Yandex Webmaster (俄罗斯): https://webmaster.yandex.com/site/ - 俄罗斯最大的搜索引擎

- Naver Search Advisor (韩国): https://searchadvisor.naver.com/console/site/summary - 韩国最大的搜索引擎，需要韩国手机号或Line账号

- Yahoo! Japan (日本): https://search.yahoo.co.jp/site/ - 日本市场份额较高的搜索引擎

## 内容建设

这部分是冷启动很关键的部分，需要对网站的主体内容进行建设，以保证搜索引的上面的搜索意图覆盖。这个量是根据业务来的，我是给的一个比较高的数额，可以根据市场进行适当的降低。

冷启动大致需要的量是：

- 500+ 纯AI刷关键词、长尾词、低难度文：主要是覆盖搜索意图。
- 80+ 核心高质量AI+人修文章：用于冲深化以及冲排名，以及提高Google对我们的评价。

在完成冷启动后，每周：

- 40+ 纯AI刷关键词、长尾词、低难度文：主要是覆盖搜索意图。
- 2+ 核心高质量AI+人修文章

## 外链建设

只有两种方式：

1. 比较慢和受控制的方式是：一家家去慢慢添加和找外链 + 社群小伙伴们做了站有一些外链积累。
2. 比较块的方式是：直接买，大概5000块，能把站的外链在一个月内就干到15+。

在完成冷启动后，就主要是去做，借大站排名冲流量以及找相关的外链。
